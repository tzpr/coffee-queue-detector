[Jetson Nano]

** redis (message broker between python script and nodejs script) **
- Install Redis using docker (https://www.docker.com/)
sudo docker run -d -p 6060:6379 --name coffee-queue redis
sudo docker start coffee-queue

** python script (capture images and do the inference) **
- Inside the inference folder
sudo python3 -m venv venv
source venv/bin/activate
sudo python3 -m pip install -r requirements.txt
python3 queue-inference-jetson-client.py

** nodejs script (slack integration api) **
- Install nodejs (https://nodejs.org/en/)
- Create .env file in slack-integration folder and put the listed variables in it:
   CLIENT_ID=id_of_the_slack_channel_app
   CLIENT_SECRET=secret_of_the_slack_channel_app
   SLACK_INTEGRATION_API_PORT=4390
   REDIS_PORT=6060
   REDIS_HOST=localhost
- Install dependencies by running:
npm install
- Start the server:
node app.js

** ngrok proxy **
Exposes the Slack integration api (localhost) to the Internet (for Slack)
- Install ngrok:
See https://ngrok.com/
- Start ngrok:
./ngrok http 4390


[Slack]
- create a channel or use an existing one
- create an app and configure a slash-command to send a request to the slack integration api 
  (via the URL generated by ngrok) and to receive the coffee queue status back.
https://api.slack.com/apps
https://api.slack.com/interactivity/slash-commands
